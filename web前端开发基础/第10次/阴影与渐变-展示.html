多重背景 & 复合属性
CSS3 支持为一个元素叠加多个背景图像；background 复合属性可一行完成所有背景设置。
多重背景图像 — 拖动滑块改变图层数量
3 层

各属性值之间用,分隔，先声明的显示在上层。

ℹ️repeat / position / size 等属性的多个值按顺序与各背景图像一一对应。
background 复合属性

将所有背景属性合并为一行，各值用空格分隔，position 和 size 之间用 / 分隔：

/* 语法格式 */
background: [color] [image] [repeat] [attachment] [position]/[size] [clip] [origin];

/* 示例：固定全屏背景图 */
body {
  background: #1a2e22 url(bg.jpg) no-repeat fixed center/cover;
}
