§ 01 什么是 HTML
定义
HTML（HyperText Markup Language，超文本标记语言）是一种用来告知浏览器如何组织页面的标记语言，它是构建网页的基础骨架。

如果把建造网页比作建房子，那么 HTML 就是砖块和框架——它规定了页面上有哪些内容、内容的层次结构是什么。HTML 本身并不是编程语言，它不包含逻辑判断，只负责描述内容的结构与语义。

三个核心知识点
  • ①
    标记 = 标签 = 元素：这三个说法在 HTML 中含义相同，都指一对（或单个）用尖括号包裹的指令符号。
  • ②
    大小写均可，建议小写：<H1> 和 <h1> 在浏览器看来是同一个标签，但业界规范建议统一使用小写，代码更整洁，团队协作更顺畅。
  • ③
    HTML 由一系列标签组成：简单说，学习 HTML 就是学习各种标签的名称、用途和写法。
💡 记住这句话：学习 HTML，本质上就是学习标签。 理解了这一点，整个 HTML 的学习就有了明确的方向。

§ 03 标签分类 点击卡片展开示例

HTML 标签按照结构可以分为两大类：双标签（大部分标签都是）和单标签（少数）。

<tag> … </tag>
双标签（成对标签）
有开始标签和结束标签，内容写在两者之间。大部分 HTML 标签都是双标签。
常见双标签：
<h1>~<h6>（标题）、<p>（段落）、<div>（容器）、<a>（链接）、<ul>（列表）
<h1>这是标题内容</h1>
<tag>
单标签（自闭合标签）
没有结束标签，没有包裹的内容，自身即为完整元素。少数标签是单标签。
常见单标签：
<hr>（水平分割线）、<br>（换行）、<img>（图片）、<input>（输入框）、<meta>（元数据）
<hr> <!-- 水平分割线，无需结束标签 -->
§ 04 文档类型声明
作用
<!DOCTYPE html> 是 HTML5 的文档类型声明，必须位于文档第一行。它告诉浏览器当前页面是使用 HTML5 规范编写的，从而以标准模式渲染页面。
html · 文档类型声明
<!-- 这必须是 HTML 文件的第一行 --> <!DOCTYPE html>
  • —
    不是 HTML 标签：<!DOCTYPE> 是一条给浏览器的"指令"，而不是 HTML 元素，不需要结束标签
  • —
    必须在第一行：放在其他任何内容（包括注释）之前
  • —
    大小写不敏感：<!DOCTYPE html> 和 <!doctype html> 效果一样，但习惯大写 DOCTYPE
⚠ 若省略 <!DOCTYPE html>，浏览器会进入"怪异模式"（Quirks Mode）——采用早期不标准的渲染方式，导致页面在不同浏览器中显示差异极大，布局混乱。
§ 05 <html> 根元素
作用
<html> 元素包裹了页面中所有的内容，是整个 HTML 文档的根容器，有时也被称为"根元素"。
html · html 元素与 lang 属性
<!-- 中文网页 --> <html lang="zh-CN"> <!-- 页面所有内容放在这里 --> </html> <!-- 英文网页 --> <html lang="en"> </html>
lang 属性的作用

lang 属性用于声明网页的主要语言，帮助浏览器、搜索引擎和辅助工具正确处理页面内容：

互动演示 · 选择语言查看 lang 值
<html lang="zh-CN">
zh-CN — 简体中文，用于中国大陆网站
  • ✓
    搜索引擎优化：Google 等搜索引擎会根据 lang 将网页分发给对应语言的用户
  • ✓
    辅助工具支持：屏幕阅读器会按照指定语言的发音规则朗读页面内容
  • ✓
    浏览器自动翻译：Chrome 等浏览器会依据 lang 判断是否提示用户翻译页面
§ 06 <head> 头部元素
作用
<head> 是头部元素，包含了文档的元（meta）数据——这些信息主要供机器（浏览器、搜索引擎）处理，不会直接显示在网页上。
HEAD 内容字符集声明 <meta charset>
html
<meta charset="UTF-8">

将文档字符集设置为 UTF-8。UTF-8 包含了绝大多数人类书面语言的字符（中文、日文、阿拉伯文……），是当今互联网最通用的编码。

⚠ 如果不加这行声明，页面在某些浏览器中可能出现乱码（尤其是含有中文内容时）。这是 head 里最重要的一行，不要遗漏！
HEAD 内容移动端适配 <meta viewport>
html
<meta name="viewport" content="width=device-width, initial-scale=1.0">

这行代码让网页在手机、平板等移动设备上以最佳状态展示——让页面宽度等于设备屏幕宽度，初始缩放比例为 1:1，避免页面在手机上缩得很小。

HEAD 内容页面标题 <title>
html
<title>我的第一个网页</title>

设置页面的标题，显示在浏览器标签页上，也是搜索引擎结果页面（SERP）中显示的标题。一个好的 title 对 SEO 非常重要。

§ 07 <body> 主体元素
作用
<body> 元素包含了页面所有会显示在浏览器中的内容，包括文本、图片、视频、音频、链接、表单等等。用户在网页上看到的一切，都来自 body。
html · body 可包含的内容示例
<body> <!-- 文本内容 --> <h1>一级标题</h1> <p>段落文本</p> <!-- 图片 --> <img src="photo.jpg" alt="一张照片"> <!-- 超链接 --> <a href="https://www.example.com">访问示例网站</a> </body>
