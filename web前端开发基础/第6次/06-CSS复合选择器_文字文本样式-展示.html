⚠️实际开发中推荐：先 list-style: none 清除符号，再通过 background-image 设置自定义图标，控制更精确。
list-style 综合属性 — 一行搞定
/* 逐一写法（不推荐，冗余） */
ul {
  list-style-type: square;
  list-style-position: inside;
}

/* 综合写法（推荐，简洁） */
ul {
  list-style: square inside;
}

/* 最常用写法：清除所有项目符号 */
ul, ol {
  list-style: none;
}
实际开发最佳实践：自定义图标列表
/* 第1步：清除默认符号 */
ul {
  list-style: none;
}

/* 第2步：腾出图标空间 */
li {
  padding-left: 20px;
}

/* 第3步：设置自定义图标 */
li {
  background-image: url("bullet.png");
  background-repeat: no-repeat;
  background-position: left center;
}
