复合选择器 · 文字与文本样式

精准定位页面元素，掌握字体与文本排版的全套控制属性
前置：HTML 结构 / CSS 基础选择器 后代 · 交集 · 并集选择器 字体属性 · 文本属性 · 列表样式
§ 01 什么是复合选择器

三种基础选择器：标签选择器、类选择器和 ID 选择器。然而，随着页面结构越来越复杂，单纯使用基础选择器会遇到很多困境——例如统一修改所有 <p> 标签时会影响到页面中所有的段落，无法做到精准区分。

选择器类型示例说明
标签选择器p选中所有 <p> 元素
类选择器.box选中所有 class 含 box 的元素
ID 选择器#title选中 id 为 title 的元素
通配选择器*选中所有元素
定义
复合选择器：将两个或多个基础选择器通过特定连接符号组合在一起，形成新的选择器。核心价值在于：在保留基础选择器灵活性的同时，大幅提高选择的精确度。
三种复合选择器总览
A B
后代选择器
Descendant
空格隔开。选中 A 内部所有层级的 B 元素，不区分嵌套深度。思维方式：位置关系。
AB
交集选择器
Intersection
无任何符号，紧密拼接。同时满足 A 和 B 两个条件，AND 关系。思维方式：条件交叉。
A, B
并集选择器
Union
逗号隔开。分别选中满足 A 或满足 B 的所有元素，OR 关系。思维方式：批量归并。
§ 02 后代选择器 Descendant Selector
定义
选中某个元素内部的后代元素。书写方式是在两个选择器之间加一个空格。不区分嵌套层级，只要满足"包含关系"就会被选中。
css · 语法格式
祖先元素 后代元素 { 属性: 值; }
交互演示 · 后代选择器命中范围
<div .article>
<p> 第一段（第一层）</p>
<section>
<p> 第二段（第二层）</p>
<div>
<p> 第三段（第三层）</p>
</div>
</section>
</div>
<p> 外部段落（.article 之外）</p>
.article p — 三层嵌套的 p 全部命中，外部 p 不受影响
css · 多级后代选择器连写
/* 选中 .nav 内部 ul 里的所有 a 链接 */ .nav ul a { text-decoration: none; color: #333; font-size: 14px; }
⚠ 工程实践建议：后代选择器层级不宜超过 3 层。层级越深，浏览器匹配时遍历节点越多，影响渲染性能，同时也会增大 HTML 结构与 CSS 的耦合度，降低可维护性。
§ 03 交集选择器 Intersection Selector · AND 关系
定义
选中同时满足多个条件的元素。多个选择器紧密拼接，中间不加任何空格。对应数学中的"集合交集"，是 AND（且） 的关系。
集合 A：所有 <p>
第一段落
第三段落
第五段落…
∩
集合 B：有 .highlight
第一段落
侧边栏标题
页脚说明…
=
结果：p.highlight
同时是 <p> 且 有 .highlight 的元素
css · 交集选择器示例
/* 既是 <p> 又有 .highlight 类 */ p.highlight { background-color: #ff0000; color: blue; } /* 三个条件的交集（较少使用） */ p.highlight.active { background-color: #00ff00; }
书写注意事项
css · 规则与常见错误
/* ✅ 规则一：标签选择器必须写在最前面 */ p.box { } /* ✅ 正确：标签 p 在前 */ .box.active { } /* ✅ 正确：纯类选择器顺序不影响 */ /* ⚠️ 规则二：绝对不能有空格！ */ p.highlight { } /* ✅ 交集：同一元素同时满足两个条件 */ p .highlight { } /* ❌ 变成后代选择器了！含义完全不同 */ /* 规则三：ID 选择器用于交集语法合法但冗余 */ div#header { } /* 合法，但直接写 #header 即可 */
🔑 关键记忆：交集选择器像"双重验证"，必须两张证件都符合条件才能通过。p.highlight 与 p .highlight 之间一个空格之差，含义天壤之别。
§ 04 并集选择器 Union Selector · OR 关系
定义
同时选中多个不同选择器对应的元素，为它们统一应用相同样式。书写方式是在选择器之间用逗号隔开。对应"集合并集"，是 OR（或） 的关系。核心价值：消除重复，统一管理公共样式。
css · 并集选择器对比（冗余 vs 简洁）
/* ❌ 冗余写法：重复代码，违背 DRY 原则 */ h1 { font-family: "黑体"; color: #2c3e50; } h2 { font-family: "黑体"; color: #2c3e50; } h3 { font-family: "黑体"; color: #2c3e50; } /* ✅ 并集选择器：每个选择器独占一行，清晰易维护 */ h1, h2, h3 { font-family: "黑体"; color: #2c3e50; } /* ✅ 混合不同类型的选择器 */ p, .intro, #footer { color: blue; font-size: 12px; }
CSS Reset 中的经典应用
css · 经典 CSS Reset 片段
html, body, h1, h2, h3, h4, h5, h6, p, ul, ol, li, figure, figcaption, blockquote, dl, dd, fieldset, textarea { margin: 0; padding: 0; box-sizing: border-box; }
✓ 推荐格式：每个选择器独占一行。可读性更强，版本控制（Git）对比更清晰，增删选择器更方便。
§ 05 字体样式属性

CSS 提供了一系列专属的字体样式属性，可以精确控制字号、字体种类、粗细、风格等维度，是网页排版的基础工具。

CSSfont-size · 字号

font-size 属性用于设置文本的字号大小，直接决定文字在页面上的显示尺寸。

单位说明
px像素单位，网页设计中最常用的单位，直观且精准
em倍率单位，相对于当前元素的父元素字体大小的倍率
%百分比单位，相对于父元素字体大小的百分比
💡推荐使用 px 单位。像素单位直观易懂，计算方便，是日常开发中的首选。
CSSfont-family · 字体

设置文本所使用的字体。浏览器按列表顺序依次查找已安装字体，找到可用字体即使用，否则继续向后查找。

css
/* 英文字体放在前，中文字体放在后 */ p { font-family: Arial, "微软雅黑", sans-serif; } /* Times New Roman 含空格须加引号 */ p { font-family: "Times New Roman", "宋体"; }
  • ①各字体之间必须用英文逗号分隔
  • ②中文字体名需加英文引号，英文字体名通常不需要
  • ③包含空格或特殊符号的字体名必须加引号
  • ④英文字体必须放在中文字体之前，尽量使用系统默认字体
CSSfont-weight · 字体粗细
属性值描述
normal默认值，标准粗细（等同于 400）
bold粗体（等同于 700），最常用
bolder / lighter比当前更粗 / 更细
100 ~ 900数值控制，400 = normal，700 = bold，数值越大越粗
CSSfont-variant & font-style · 变体与风格
属性属性值效果
font-variant
仅对英文有效		normal默认，标准显示
small-caps小写字母转为小型大写显示
font-stylenormal标准正体（可用于取消 em 的默认斜体）
italic使用字体本身的斜体版本，效果更自然（推荐）
oblique通过算法将正体倾斜，字形略显生硬
CSS · 复合font · 综合设置字体样式
语法格式
font: font-style   font-weight   font-size/line-height   font-family;
css
/* 逐一书写（代码较多） */ p { font-style: italic; font-weight: bold; font-size: 16px; line-height: 1.8; font-family: "微软雅黑"; } /* ✅ 等效的综合写法（代码简洁） */ p { font: italic bold 16px/1.8 "微软雅黑"; }
  • ①属性书写顺序严格固定，不能随意调换
  • ②font-size 和 font-family 是必填项，缺一则整条声明失效
CSS3@font-face · 服务器字体

允许开发者加载服务器上的自定义字体文件，使网页不受限于用户本地已安装的字体。

css · @font-face 三步使用
/* 第一步：下载字体文件，存放到 fonts/ 文件夹 */ /* 第二步：使用 @font-face 定义服务器字体 */ @font-face { font-family: "ZiTi"; /* 自定义字体名称 */ src: url("fonts/ZiTi.ttf"); /* 字体文件路径 */ } /* 多格式兼容写法（推荐） */ @font-face { font-family: "MyFont"; src: url("fonts/myfont.woff2") format("woff2"), url("fonts/myfont.woff") format("woff"), url("fonts/myfont.ttf") format("truetype"); } /* 第三步：应用到页面元素 */ h1 { font-family: "ZiTi"; }
§ 11 文本外观属性
CSScolor · 文本颜色

定义文本内容的前景颜色，支持三种表示方式：

css · 颜色三种写法
/* ① 预定义颜色关键字 */ p { color: red; } /* ② 十六进制（最常用），可缩写：#FF6600 → #F60 */ p { color: #FF6600; } p { color: #F60; } /* 等效缩写 */ /* ③ RGB 代码 */ p { color: rgb(255, 0, 0); } p { color: rgb(100%, 0%, 0%); } /* 百分比写法，0 不能省略 % */
CSS字间距 / 单词间距 / 行高
属性作用说明
letter-spacing字符间距对中英文均有效，支持负值
word-spacing单词间距仅对英文有效（以空格分隔的单词），对中文无效
line-height行间距（行高）推荐 px 或纯数字（如 1.6）；常用技巧：行高 = 元素高度可实现单行文字垂直居中
css · line-height 垂直居中技巧
.button { height: 40px; line-height: 40px; /* 行高等于高度，实现单行文字垂直居中 */ }
CSStext-transform · 英文大小写转换
属性值效果
none默认，不转换
capitalize每个单词首字母大写
uppercase全部大写（常用于导航菜单、按钮）
lowercase全部小写
CSStext-decoration · 文本装饰线
属性值效果
none无装饰（常用于去除 <a> 标签的默认下划线）
underline下划线
overline上划线
line-through删除线（适用于原价、已完成项）
💡多值叠加：text-decoration: underline line-through; 可同时设置下划线和删除线，值之间用空格分隔。
CSStext-align / text-indent · 对齐与缩进
属性属性值说明
text-align
仅对块级元素有效		left左对齐（默认）
center居中对齐
right右对齐
text-indent2em推荐：缩进两个汉字宽度，随字体大小自动调整
24px固定像素缩进
-2em负值：悬挂缩进（需配合 padding-left: 2em）
⚠text-align 对行内元素（<img>、<span>）无效。若需图像居中，应为图像的父容器设置 text-align: center。
CSS3text-shadow · 文字阴影效果
语法格式
text-shadow: h-shadow   v-shadow   blur   color;
参数含义允许负值
h-shadow水平偏移距离，正值向右，负值向左✓
v-shadow垂直偏移距离，正值向下，负值向上✓
blur模糊半径，值越大边缘越模糊✗ 只能正值
color阴影颜色—
交互演示 · 阴影效果切换
文字阴影效果演示
text-shadow: 2px 2px 4px #888888
css · 常用阴影写法
h1 { text-shadow: 2px 2px 4px #888888; } /* 基础阴影 */ .sharp { text-shadow: 3px 3px 0 #000; } /* 清晰无模糊 */ .glow { text-shadow: 0 0 10px rgba(0,150,255,0.8); } /* 发光 */ /* 多阴影叠加（逗号分隔） */ p { text-shadow: 10px 10px 10px red, 20px 20px 20px green; }
CSS3text-overflow / word-wrap · 溢出文本处理
文字溢出省略号效果演示
无处理（溢出显示）
这是一段很长的文字内容，超出容器宽度不处理
text-overflow: ellipsis（省略号）
这是一段很长的文字内容，超出容器宽度后显示省略号
🔑
单行省略号的四步标准写法（必须掌握）：
css · 单行文字溢出省略标准写法
.card-title { width: 200px; /* ① 限制宽度 */ white-space: nowrap; /* ② 强制不换行 */ overflow: hidden; /* ③ 隐藏溢出内容 */ text-overflow: ellipsis; /* ④ 显示省略号 */ } /* word-wrap：强制长单词/URL 内部换行 */ .url-display { word-wrap: break-word; width: 200px; }
§ 18 列表样式属性

CSS 提供列表样式属性，通过样式表控制列表的外观，符合"结构与样式相分离"的原则。

交互演示 · list-style-type 效果
无序列表 <ul>
  • 列表项一
  • 列表项二
  • 列表项三
有序列表 <ol>
  1. 列表项一
  2. 列表项二
  3. 列表项三
list-style-type: disc — 实心圆，无序列表默认值
属性作用常用值
list-style-type项目符号类型none、disc、circle、square、decimal
list-style-image自定义图片符号url("icon.png")（实际开发不推荐，改用 background-image）
list-style-position符号相对文本的位置outside（默认，符号在文本外侧）、inside（符号在文本内侧）
list-style复合属性list-style: none;（实际开发最常用）
实际开发最佳实践
css · 自定义列表图标标准做法
/* 第一步：去除所有默认项目符号 */ ul, ol { list-style: none; } /* 第二步：腾出左侧空间 */ li { padding-left: 20px; } /* 第三步：通过 background-image 设置自定义图标（比 list-style-image 更灵活） */ li { padding-left: 20px; background-image: url("images/bullet.png"); background-repeat: no-repeat; background-position: left center; }
