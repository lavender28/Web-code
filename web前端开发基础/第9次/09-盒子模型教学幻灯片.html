认识盒子模型

什么是盒子模型？

  • CSS 将 HTML 页面中的每一个元素都看作一个矩形盒子
  • 每个盒子占据一定的页面空间
  • 一个 HTML 页面由无数盒子排列、堆叠、嵌套而成

学习网页布局的两个核心任务：

① 理解一个独立盒子的内部结构

② 掌握多个盒子之间的相互关系

页面结构示意

导航菜单
广告区
文章正文区域
页面底部
💡

<div></div> 是最常用的"盒子"标签，本身无默认样式，可自由定义大小、颜色、位置。

认识盒子模型

盒子尺寸的计算

实际宽度 = width
+ padding（左+右）
+ border（左+右）
+ margin（左+右）
⚠️

width / height 仅指内容区域（content）的宽高，不包含 padding、border、margin。

div {
  width: 200px; /* 内容宽度 */
  height: 100px;
  padding: 10px; /* 四边各10px */
  border: 2px solid red;
  margin: 20px;
}

实际宽度 = 200 + 10×2 + 2×2 + 20×2 = 264px

实际高度 = 100 + 10×2 + 2×2 + 20×2 = 164px

💡

width/height 仅适用于块元素，对行内元素无效。<img>、<input> 为特殊例外。

边框的设置

边框样式（border-style）

⚑ 边框样式是显示边框的前提条件，不设置则边框不显示。

属性值效果
none无边框（默认值）
solid单实线 ── 最常用
dashed虚线 - - - - -
dotted点线 · · · · ·
double双实线 ═══════
⚠️

值复制原则：1值=四边，2值=上下/左右，3值=上/左右/下，4值=上右下左（顺时针）

border-style: double; /* 四边双实线 */

border-style: solid dashed;
/* 上下单实线，左右虚线 */

border-style: solid dashed dotted;
/* 上实线，左右虚线，下点线 */

单边属性（更精确控制）：

border-top-style: solid;
border-right-style: dashed;
border-bottom-style: dotted;
border-left-style: double;

边框的设置

宽度 / 颜色 / 综合设置

border-width

border-width: 1px; /* 四边 */
border-width: 2px 1px;
/* 上下2px，左右1px */
粗细场景
1px表格线、输入框
2~3px卡片、按钮
4px+装饰性粗边框

border-color

/* 三种颜色写法 */
border-color: red;
border-color: #f00;
border-color: rgb(255,0,0);

/* 十六进制简写 */
#FF0000 → #f00
#FFFFFF → #fff
💡

当每两位相同时可简写为一位

border 综合属性

/* 一行搞定三要素 */
border: 3px solid #00f;

/* 单侧综合属性 */
border-top: 2px solid red;
border-right: 3px double #f90;
border-bottom: 2px dotted #c0f;
border-left: 3px double #f90;
⚠️

style 不能省略，否则边框不显示

07 / 13

边框的设置

圆角边框（border-radius）

原理：在每个角落用一段圆弧替代直角，圆弧所在圆的半径即 border-radius 值。

border-radius: 参数1 / 参数2;
/* 参数1=水平半径，参数2=垂直半径 */
/* 省略参数2则等于参数1 */

border-radius: 50px; /* 四角相同 */
border-radius: 50px / 30px; /* 椭圆弧 */
值个数对应位置
1个值四个角全部相同
2个值左上+右下 / 右上+左下
3个值左上 / 右上+左下 / 右下
4个值左上 右上 右下 左下（顺时针）

实用技巧演示：

50%
border-radius: 50%;
/* 正圆（宽=高） */
胶囊按钮
border-radius: 20px;
/* = 高度的一半 */
💡

等宽等高 + border-radius: 50% = 正圆；高度一半 = 胶囊形按钮

08 / 13

内边距的设置

padding — 内容与边框之间的距离

关键特性 ①：内边距区域继承背景色，background-color 会延伸到 padding 区域。

关键特性 ②：内边距会撑大盒子，必须计入实际尺寸。

⚠️

padding 不允许负值。百分比基准是父元素的宽度（width），与高度无关。

/* 值复制原则 */
padding: 10px; /* 四边均为10px */
padding: 10px 20px; /* 上下/左右 */
padding: 10px 20px 30px; /* 上/左右/下 */
padding: 10px 20px 30px 40px;
#box {
  width: 400px;
  border: 5px solid red;
  padding: 10px 40px 80px 120px;
  /* 上10，右40，下80，左120 */
  background-color: #e8f4fd;
  /* 背景色延伸到内边距区域 */
}

单边属性：

padding-top: 10px;
padding-right: 20px;
padding-bottom: 30px;
padding-left: 40px;
盒子实际宽度 = 400 + 40×2 + 5×2 = 490px

外边距的设置

margin — 盒子与外部世界的距离

对比维度padding（内）margin（外）
位置边框内侧边框外侧
背景色✅ 会显示❌ 始终透明
负值❌ 不允许✅ 允许
合并不会发生上下方向会合并
* {
  padding: 0; /* 清除默认内边距 */
  margin: 0; /* 清除默认外边距 */
}

上下外边距合并（塌陷）

两个垂直相邻的块元素，各自设置上下外边距时，不会叠加，而是合并取较大值。

box1: margin-bottom = 50px
box2: margin-top = 30px
─────────────────────────
实际间距 = max(50, 30) = 50px
不是 80px！
/* 解决方案 */
overflow: hidden; /* 触发BFC，推荐 */
border: 1px solid transparent;
padding-top: 1px;
10 / 13

外边距的设置

水平居中 · 负值 · 默认值重置

水平居中 常用

.container {
  width: 960px;
  margin: 0 auto;
  /* 左右auto → 均分余量 */
}
💡

auto = 浏览器自动均分，仅对块元素有效

负值效果 高级

#box2 {
  margin-top: -20px;
  /* 向上移动20px */
  /* 与上方元素重叠 */
}
⚠️

初学阶段谨慎使用，易造成布局混乱

重置默认样式 必备

* {
  padding: 0;
  margin: 0;
}

不同浏览器默认样式不同，项目开始时必须先重置，保证跨浏览器一致性。

📖

进阶项目推荐使用 Eric Meyer's Reset CSS 或 Normalize.css 进行更精细的默认样式控制，而非简单的全部清零。

宽度与高度

width / height / box-sizing

width / height 注意事项

  • 仅定义内容区域（content）的大小，不含 padding 和 border
  • 仅适用于块级元素，行内元素无效
  • <img> 和 <input> 虽为行内元素，但可以设置宽高
  • 百分比宽度参照父元素的内容宽度
width: 500px; /* 像素值（最常用）*/
width: 50%; /* 相对父元素宽度 */
width: auto; /* 自动填满父元素 */

box-sizing CSS3

/* 默认：width仅指内容区域 */
box-sizing: content-box;

/* width 包含 padding + border */
box-sizing: border-box; ← 推荐

使用 border-box 后，width: 300px 就是盒子的真实宽度，无需手动减去 padding 和 border。

/* 全局推荐写法 */
*, *::before, *::after {
  box-sizing: border-box;
}

知识点总览

01 · 盒子模型概念

content + padding + border + margin 四层结构；尺寸计算公式

02 · <div> 标签

最基础的布局容器，无语义，与 id/class 配合，优先语义标签

03 · 边框设置

style/width/color 三要素；border 综合属性；border-radius 圆角

不能为负值；背景色延伸到此区域；百分比基准=父元素宽度

05 · margin 外边距

可为负值；上下方向外边距合并（取较大值）；0 auto 水平居中

06 · box-sizing

border-box 让 width 包含 padding+border，现代项目全局推荐
