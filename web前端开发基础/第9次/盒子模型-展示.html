📄
content（内容）
由 width / height 控制大小，承载文字、图片等实际内容。
🟩
padding（内边距）
内容与边框之间的空隙，背景色延伸到此区域，不能为负值。
🟥
border（边框）
包裹内边距和内容的可见线框，可设置样式、宽度、颜色。
🟨
margin（外边距）
盒子与相邻元素间的透明距离，不显示背景色，允许负值。
尺寸计算公式
实际宽度 = width + padding×2 + border×2 + margin×2 = …
实际高度 = height + padding×2 + border×2 + margin×2 = …
⚠️width / height 仅指 content 区域，实际占用空间还需加上 padding、border、margin。

⚠️ 水平方向外边距直接相加，无合并规则。负值使盒子向左移动，产生重叠（红色区域）。
⚠️ 垂直外边距合并 — 取较大值而非相加
误以为：直接相加
box1 — margin-bottom: 50px
50 + 30 = 80px ✗
box2 — margin-top: 30px
间距 = 80px ✗
实际结果：取较大值 ✓
box1 — margin-bottom: 50px
max(50, 30) = 50px
box2 — margin-top: 30px
间距 = 50px ✓
